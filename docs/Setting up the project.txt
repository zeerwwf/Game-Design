* JDK

Download the jdk on http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
Install the jdk
On Eclipse :
Windows > Preferences > Java > Installed JREs > Add > Standard VM > Directory ... > Search your jdk (should be in C:\Program Files\java\...) > Finish > Check the jdk not the jre (3)
You can now develop on eclipse

* Clone the project

File > New > Other > Check out Maven Projects from SCM (1)
Click on m2e Marketplace (2) and install m2e egit
After Eclipse restarted do again File > New > Other > Check out Maven Projects from SCM
Fill the scm url field with the url given by bitbucket when clicking on Clone (choose https not ssh) > Next > Provide your credentials and wait :-)
